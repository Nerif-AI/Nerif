## 1. Async Agent

### Problem
`NerifAgent.run()` is sync-only. Users with async codebases must wrap it in `run_in_executor`, losing the benefit of non-blocking I/O that `SimpleChatModel.achat()` already provides.

### Design

Add `async def arun(message: str) -> str` to `NerifAgent`. It mirrors `run()` exactly but calls `self.model.achat()` instead of `self.model.chat()`.

Tool execution remains sync by default (user-defined functions are typically sync). To support async tools, add an optional `async_func` field to `Tool`:

```python
# agent/tool.py — preserves existing field name `func` for backward compatibility
class Tool:
    def __init__(
        self,
        name: str,
        description: str,
        parameters: Dict[str, Any],
        func: Callable[..., Any],              # existing field, unchanged
        async_func: Optional[Callable] = None,  # NEW: optional async executor
    ):
        self.name = name
        self.description = description
        self.parameters = parameters
        self.func = func
        self.async_func = async_func

    # existing methods unchanged: to_openai_tool(), execute()

    async def aexecute(self, **kwargs) -> Any:
        """Async tool execution. Uses async_func if available, else falls back to sync func."""
        if self.async_func is not None:
            return await self.async_func(**kwargs)
        return self.func(**kwargs)
```

**Backward Compatibility**: The `func` field name is preserved. The `async_func` field has a default of `None`, so all existing `Tool(name=..., description=..., parameters=..., func=...)` calls continue to work unchanged.

Agent changes:
```python
# agent/agent.py
async def _execute_tool_call_async(self, tool_call: ToolCallResult) -> str:
    """Like _execute_tool_call but awaits async tools."""
    tool = self.tools.get(tool_call.name)
    if tool is None:
        return json.dumps({"error": f"Tool '{tool_call.name}' not found"})
    try:
        args = json.loads(tool_call.arguments)
    except json.JSONDecodeError:
        return json.dumps({"error": f"Invalid JSON arguments: {tool_call.arguments}"})
    try:
        result = await tool.aexecute(**args)
        return result if isinstance(result, str) else json.dumps(result, default=str)
    except Exception as e:
        return json.dumps({"error": str(e)})

async def arun(self, message: str) -> str:
    """Async version of run(). Uses achat() and aexecute().

    When multiple tool calls are returned in a single response,
    they are executed concurrently via asyncio.gather().
    """
    tool_dicts = self._get_tool_dicts() if self.tools else None
    result = await self.model.achat(message, append=True, tools=tool_dicts,
                                     tool_choice="auto" if tool_dicts else None)

    for _ in range(self.max_iterations):
        if isinstance(result, str):
            return result
        if not isinstance(result, list):
            return str(result)

        # Execute tool calls concurrently when there are multiple
        if len(result) > 1:
            tool_results = await asyncio.gather(
                *[self._execute_tool_call_async(tc) for tc in result]
            )
            for tool_call, tool_result in zip(result, tool_results):
                self.model.messages.append({
                    "role": "tool", "tool_call_id": tool_call.id, "content": tool_result,
                })
        else:
            # Single tool call — execute directly
            tool_result = await self._execute_tool_call_async(result[0])
            self.model.messages.append({
                "role": "tool", "tool_call_id": result[0].id, "content": tool_result,
            })

        result = await self.model.achat("", append=True, tools=tool_dicts,
                                         tool_choice="auto" if tool_dicts else None)

    LOGGER.warning("Agent reached max_iterations (%d) without producing a final response", self.max_iterations)
    return result if isinstance(result, str) else "Agent reached maximum iterations without a final response."
```

The `@tool` decorator gains an optional `async_func` parameter. **Existing positional arguments remain required** to preserve backward compatibility:
```python
def tool(name: str, description: str, parameters: Dict[str, Any], async_func=None):
    """Decorator to create a Tool from a function.

    Args:
        name: Tool name (required).
        description: Tool description (required).
        parameters: JSON schema for parameters (required).
        async_func: Optional async version of the function.
    """
    def decorator(func: Callable[..., Any]) -> Tool:
        return Tool(name=name, description=description, parameters=parameters,
                    func=func, async_func=async_func)
    return decorator
```

### Exports
- `NerifAgent.arun` — new method
- `Tool.async_func`, `Tool.aexecute` — new field/method

## 2. Model Fallback Chain

### Problem
When a primary model is rate-limited or unavailable, the entire request fails. Users want automatic failover to alternative models.

### Design

New file `utils/fallback.py`:

```python
from dataclasses import dataclass, field
from typing import Set

from .retry import _is_transient_error  # shared helper (see below)


@dataclass
class FallbackConfig:
    """Configure model fallback behavior."""
    models: list                         # ordered list: primary first, then fallbacks
    fallback_on: Set[int] = field(       # HTTP status codes that trigger fallback
        default_factory=lambda: {429, 500, 502, 503, 504}
    )
    fallback_on_timeout: bool = True     # also fallback on timeouts

    def should_fallback(self, exception: Exception) -> bool:
        return _is_transient_error(exception, self.fallback_on, self.fallback_on_timeout)
```

**Shared helper** — extracted to `utils/retry.py` to eliminate duplication between `RetryConfig.should_retry()` and `FallbackConfig.should_fallback()`:

```python
# utils/retry.py — new helper function
def _is_transient_error(exception: Exception, status_codes: Set[int], on_timeout: bool) -> bool:
    """Check if an exception represents a transient error worth retrying/falling back from."""
    if isinstance(exception, httpx.TimeoutException):
        return on_timeout
    if isinstance(exception, httpx.HTTPStatusError):
        return exception.response.status_code in status_codes
    if isinstance(exception, (httpx.ConnectError, httpx.RemoteProtocolError)):
        return True
    return False

class RetryConfig:
    # ... existing fields ...
    def should_retry(self, attempt: int, exception: Exception) -> bool:
        if attempt >= self.max_retries:
            return False
        return _is_transient_error(exception, self.retryable_status_codes, self.retry_on_timeout)
```

Integration into `SimpleChatModel`:
```python
class SimpleChatModel:
    def __init__(self, ..., fallback: Optional[List[str]] = None):
        self.fallback_config = None
        if fallback:  # truthy check — empty list creates no FallbackConfig
            self.fallback_config = FallbackConfig(models=[model] + fallback)
```

The fallback logic wraps the existing `get_model_response` / `get_model_response_async` calls in `chat()` / `achat()`:

```python
def _call_with_fallback(self, call_fn, kwargs):
    """Try primary model, then each fallback in order."""
    if self.fallback_config is None:
        return call_fn(**kwargs)

    last_exception = None
    for model_name in self.fallback_config.models:
        try:
            kwargs["model"] = model_name
            return call_fn(**kwargs)
        except Exception as e:
            last_exception = e
            if not self.fallback_config.should_fallback(e):
                raise
            LOGGER.warning("Model %s failed, trying next fallback: %s", model_name, e)
    raise last_exception

async def _call_with_fallback_async(self, call_fn, kwargs):
    """Async version of _call_with_fallback."""
    if self.fallback_config is None:
        return await call_fn(**kwargs)

    last_exception = None
    for model_name in self.fallback_config.models:
        try:
            kwargs["model"] = model_name
            return await call_fn(**kwargs)
        except Exception as e:
            last_exception = e
            if not self.fallback_config.should_fallback(e):
                raise
            LOGGER.warning("Model %s failed, trying next fallback: %s", model_name, e)
    raise last_exception
```

The fallback interacts cleanly with retry: each model attempt uses the configured `retry_config` before moving to the next fallback. The order is: retry model A → fallback to model B → retry model B → fallback to model C → etc.

`NerifAgent` inherits fallback behavior automatically through its `SimpleChatModel` instance:
```python
class NerifAgent:
    def __init__(self, ..., fallback: Optional[List[str]] = None):
        self.model = SimpleChatModel(..., fallback=fallback)
```

### Exports
- `FallbackConfig` from `utils/fallback.py`
- `SimpleChatModel(fallback=[...])` — new parameter
- `NerifAgent(fallback=[...])` — new parameter

## 3. Streaming Enhancements

### Problem
`stream_chat` and `astream_chat` already save the completed response to memory (v1.2.0). However, they lack:
- Token counter integration (no cost tracking for streamed responses)
- Retry config support
- Fallback support
- Callback firing

### Design

Enhance both `stream_chat` **and** `astream_chat` to pass `counter` and `retry_config` to the streaming call, and fire callbacks. Both methods receive identical enhancements.

```python
def stream_chat(self, message, append=False, max_tokens=None, response_format=None):
    # ... existing message setup ...
    kwargs = {
        "model": self.model, "temperature": self.temperature, "max_tokens": req_max_tokens,
    }
    if response_format is not None:
        kwargs["response_format"] = response_format
    # NEW: pass counter for token tracking
    if self.counter is not None:
        kwargs["counter"] = self.counter
    # NEW: pass retry_config
    if self.retry_config is not None:
        kwargs["retry_config"] = self.retry_config

    full_response = []
    for chunk in get_model_response_stream(self.messages, **kwargs):
        if chunk.content:
            full_response.append(chunk.content)
            yield chunk.content

    complete_text = "".join(full_response)
    if append:
        if self.memory is not None:
            self.memory.add_message("assistant", complete_text)
        else:
            self.messages.append({"role": "assistant", "content": complete_text})
    else:
        self.reset()
```

`astream_chat` receives the same changes (using `get_model_response_stream_async`).

The streaming functions in `utils.py` (`get_model_response_stream` and `get_model_response_stream_async`) will be updated to:
1. Accept an optional `counter` parameter
2. Track usage from the final SSE chunk (OpenAI sends usage in `stream_options: {include_usage: true}`)
3. Accept `retry_config` for retrying the initial connection

## 4. Callback / Hook System

### Problem
Users need visibility into LLM operations for logging, monitoring, tracing, and debugging. The existing `NerifTokenCounter` has basic callbacks (`on_request_start`, `on_request_end`, `on_error`) but they're limited to the counter object and don't cover tool calls, fallback switches, or memory operations.

### Design

New file `utils/callbacks.py`:

```python
import logging
from dataclasses import dataclass
from typing import Any, Dict, List, Optional

_CALLBACK_LOGGER = logging.getLogger("nerif.callbacks")


@dataclass
class LLMStartEvent:
    """Fired before an LLM API call."""
    model: str
    messages: List[Dict]
    timestamp: float
    kwargs: Dict[str, Any]

@dataclass
class LLMEndEvent:
    """Fired after a successful LLM API call."""
    model: str
    response: str
    latency_ms: float
    prompt_tokens: int
    completion_tokens: int
    cost_usd: float  # 0.0 when counter is not available

@dataclass
class LLMErrorEvent:
    """Fired when an LLM API call fails."""
    model: str
    error: Exception
    latency_ms: float
    will_retry: bool

@dataclass
class ToolCallEvent:
    """Fired when an agent executes a tool."""
    tool_name: str
    arguments: Dict[str, Any]
    result: str
    latency_ms: float
    success: bool

@dataclass
class FallbackEvent:
    """Fired when a model fallback occurs."""
    failed_model: str
    next_model: str
    error: Exception

@dataclass
class RetryEvent:
    """Fired when a request is being retried."""
    model: str
    attempt: int
    max_retries: int
    delay: float
    error: Exception

@dataclass
class MemoryEvent:
    """Fired on memory operations (summarize, window trim)."""
    action: str          # "summarize", "trim", "clear"
    messages_before: int
    messages_after: int
    summary: Optional[str] = None


class CallbackHandler:
    """Base class for callback handlers. Override methods you care about."""

    def on_llm_start(self, event: LLMStartEvent) -> None:
        pass

    def on_llm_end(self, event: LLMEndEvent) -> None:
        pass

    def on_llm_error(self, event: LLMErrorEvent) -> None:
        pass

    def on_tool_call(self, event: ToolCallEvent) -> None:
        pass

    def on_fallback(self, event: FallbackEvent) -> None:
        pass

    def on_retry(self, event: RetryEvent) -> None:
        pass

    def on_memory(self, event: MemoryEvent) -> None:
        pass


class CallbackManager:
    """Manages a list of callback handlers."""

    def __init__(self):
        self._handlers: List[CallbackHandler] = []

    def add_handler(self, handler: CallbackHandler) -> None:
        self._handlers.append(handler)

    def remove_handler(self, handler: CallbackHandler) -> None:
        self._handlers.remove(handler)

    def fire(self, method_name: str, event: Any) -> None:
        for handler in self._handlers:
            fn = getattr(handler, method_name, None)
            if fn is not None:
                try:
                    fn(event)
                except Exception:
                    _CALLBACK_LOGGER.debug(
                        "Callback %s.%s raised an exception",
                        type(handler).__name__, method_name, exc_info=True,
                    )
```

**Note on `cost_usd` in `LLMEndEvent`**: The cost is computed via `NerifTokenCounter._calculate_cost()` when a `counter` is available on `SimpleChatModel`. When `counter is None`, `cost_usd` is set to `0.0`. This avoids duplicating pricing logic — the callback system reports cost only when observability is enabled.

**Convenience: LoggingCallbackHandler**

A built-in handler that logs all events via Python `logging`:
```python
class LoggingCallbackHandler(CallbackHandler):
    def __init__(self, logger=None):
        self.logger = logger or logging.getLogger("nerif.callbacks")

    def on_llm_start(self, event):
        self.logger.info("LLM request to %s (%d messages)", event.model, len(event.messages))

    def on_llm_end(self, event):
        self.logger.info("LLM response from %s (%.0fms, $%.6f)", event.model, event.latency_ms, event.cost_usd)

    def on_llm_error(self, event):
        self.logger.warning("LLM error from %s: %s (%.0fms, retry=%s)",
                           event.model, event.error, event.latency_ms, event.will_retry)

    def on_tool_call(self, event):
        self.logger.info("Tool %s called (%.0fms, success=%s)", event.tool_name, event.latency_ms, event.success)

    def on_fallback(self, event):
        self.logger.warning("Fallback: %s -> %s due to %s", event.failed_model, event.next_model, event.error)

    def on_retry(self, event):
        self.logger.info("Retry %d/%d for %s (delay=%.1fs)", event.attempt, event.max_retries, event.model, event.delay)

    def on_memory(self, event):
        self.logger.info("Memory %s: %d -> %d messages", event.action, event.messages_before, event.messages_after)
```

**Integration Points:**

1. `SimpleChatModel.__init__` accepts optional `callbacks: CallbackManager`
2. `SimpleChatModel.chat()` / `achat()` / `stream_chat()` / `astream_chat()` fire `on_llm_start` before the API call and `on_llm_end` / `on_llm_error` after
3. `NerifAgent.__init__` accepts optional `callbacks: CallbackManager`, passes it to `SimpleChatModel`, and fires `on_tool_call` during tool execution
4. Fallback logic in `_call_with_fallback` fires `on_fallback` when switching models
5. Retry logic in `utils/retry.py` accepts optional `callbacks` and fires `on_retry` on each retry attempt
6. `ConversationMemory` accepts optional `callbacks` and fires `on_memory` during summarization and trimming

### Backward Compatibility
Existing `NerifTokenCounter.on_request_start/on_request_end/on_error` callbacks remain functional. The new callback system is additive — users can use either or both.

## 5. Prompt Template

### Problem
Users concatenate strings to build prompts, which is error-prone and hard to maintain.

### Design

New file `utils/prompt.py`:

```python
import re
from typing import Any, Dict, Optional, Set


class PromptTemplate:
    """Lightweight prompt template with variable substitution.

    Usage:
        tpl = PromptTemplate("Summarize {text} in {language}")
        prompt = tpl.format(text="Hello world", language="Chinese")
        # -> "Summarize Hello world in Chinese"

        # With defaults:
        tpl = PromptTemplate("Answer in {language}", defaults={"language": "English"})
        prompt = tpl.format()  # "Answer in English"
        prompt = tpl.format(language="Chinese")  # "Answer in Chinese"

        # Conditional sections (gated on a single trigger variable):
        tpl = PromptTemplate("Do X.{? Use format: {format}}")
        prompt = tpl.format()  # "Do X."
        prompt = tpl.format(format="JSON")  # "Do X. Use format: JSON"

    Conditional sections are gated on one trigger variable. If present,
    the entire section is rendered. Any other variables inside the section
    use silent substitution (empty string if missing). This differs from
    top-level variables which raise KeyError if missing.
    """

    _VAR_PATTERN = re.compile(r"\{(\w+)\}")
    _CONDITIONAL_PATTERN = re.compile(r"\{\?([^}]*\{(\w+)\}[^}]*)\}")

    def __init__(self, template: str, defaults: Optional[Dict[str, Any]] = None):
        self.template = template
        self.defaults = defaults or {}

    @property
    def variables(self) -> Set[str]:
        """Return all variable names in the template (including inside conditionals)."""
        return set(self._VAR_PATTERN.findall(self.template))

    def format(self, **kwargs) -> str:
        """Render the template with the given variables.

        Raises:
            KeyError: If a required top-level variable is missing.
                      Variables inside conditional sections use silent
                      substitution (empty string) instead.
        """
        merged = {**self.defaults, **kwargs}

        # Process conditional sections first
        def replace_conditional(match):
            section = match.group(1)
            var_name = match.group(2)
            if var_name in merged and merged[var_name] is not None:
                # Render section; variables inside use silent substitution
                return self._VAR_PATTERN.sub(lambda m: str(merged.get(m.group(1), "")), section)
            return ""

        result = self._CONDITIONAL_PATTERN.sub(replace_conditional, self.template)

        # Then substitute remaining top-level variables (strict — raises KeyError)
        def replace_var(match):
            name = match.group(1)
            if name not in merged:
                raise KeyError(f"Missing template variable: {name}")
            return str(merged[name])

        return self._VAR_PATTERN.sub(replace_var, result)

    def partial(self, **kwargs) -> "PromptTemplate":
        """Return a new template with some variables pre-filled as defaults."""
        new_defaults = {**self.defaults, **kwargs}
        return PromptTemplate(self.template, defaults=new_defaults)

    def __repr__(self) -> str:
        return f"PromptTemplate({self.template!r})"

    def __add__(self, other: "PromptTemplate") -> "PromptTemplate":
        """Concatenate two templates."""
        combined = self.template + other.template
        merged_defaults = {**self.defaults, **other.defaults}
        return PromptTemplate(combined, defaults=merged_defaults)
```

**Why not Jinja2?** Nerif's design principle is minimal dependencies (only `httpx` for core). A simple regex-based template covers 95% of use cases without adding a dependency.

### Exports
- `PromptTemplate` from `utils/prompt.py`, re-exported via `utils/__init__.py`

## 6. Rate Limiting / Concurrency Control

### Problem
Users making many concurrent requests (especially in async code) can trigger provider rate limits (429 errors). Even with retry, this wastes time and money.

### Design

New file `utils/rate_limit.py`:

```python
import asyncio
import threading
import time
from dataclasses import dataclass
from typing import Dict, Optional


@dataclass
class RateLimitConfig:
    """Rate limit configuration for a model or provider.

    Args:
        requests_per_minute: Max requests per minute (0 = unlimited).
        requests_per_second: Max requests per second (0 = unlimited). Takes precedence.
        max_concurrent: Max concurrent requests (0 = unlimited).
    """
    requests_per_minute: int = 0
    requests_per_second: float = 0.0
    max_concurrent: int = 0

    @property
    def min_interval(self) -> float:
        """Minimum seconds between requests."""
        if self.requests_per_second > 0:
            return 1.0 / self.requests_per_second
        if self.requests_per_minute > 0:
            return 60.0 / self.requests_per_minute
        return 0.0


class RateLimiter:
    """Thread-safe rate limiter using interval enforcement + semaphore.

    Uses separate locks for sync (threading.Lock) and async (asyncio.Lock)
    paths to avoid holding a threading lock across await points.
    """

    def __init__(self, config: RateLimitConfig):
        self.config = config
        self._sync_lock = threading.Lock()
        self._async_lock: Optional[asyncio.Lock] = None
        self._last_request = 0.0
        self._semaphore: Optional[threading.Semaphore] = None
        self._async_semaphore: Optional[asyncio.Semaphore] = None
        if config.max_concurrent > 0:
            self._semaphore = threading.Semaphore(config.max_concurrent)
            self._async_semaphore = asyncio.Semaphore(config.max_concurrent)

    def acquire(self) -> None:
        """Block until we can make a request (sync)."""
        if self._semaphore is not None:
            self._semaphore.acquire()

        interval = self.config.min_interval
        if interval > 0:
            with self._sync_lock:
                now = time.monotonic()
                elapsed = now - self._last_request
                if elapsed < interval:
                    time.sleep(interval - elapsed)
                self._last_request = time.monotonic()

    def release(self) -> None:
        """Release a concurrent request slot (sync)."""
        if self._semaphore is not None:
            self._semaphore.release()

    async def aacquire(self) -> None:
        """Block until we can make a request (async).

        Uses asyncio.Lock to avoid holding a threading lock across await.
        """
        if self._async_semaphore is not None:
            await self._async_semaphore.acquire()

        interval = self.config.min_interval
        if interval > 0:
            if self._async_lock is None:
                self._async_lock = asyncio.Lock()
            async with self._async_lock:
                now = time.monotonic()
                elapsed = now - self._last_request
                if elapsed < interval:
                    await asyncio.sleep(interval - elapsed)
                self._last_request = time.monotonic()

    def arelease(self) -> None:
        """Release a concurrent request slot (async).

        Note: This is a sync method (not a coroutine) because
        asyncio.Semaphore.release() is synchronous. Named `arelease`
        for API symmetry with `aacquire`.
        """
        if self._async_semaphore is not None:
            self._async_semaphore.release()


class RateLimiterRegistry:
    """Global registry of per-model/provider rate limiters."""

    def __init__(self):
        self._limiters: Dict[str, RateLimiter] = {}
        self._lock = threading.Lock()

    def configure(self, key: str, config: RateLimitConfig) -> None:
        """Set rate limit for a model name or provider prefix."""
        with self._lock:
            self._limiters[key] = RateLimiter(config)

    def get(self, model: str) -> Optional[RateLimiter]:
        """Get limiter for a model. Checks exact match, then provider prefix."""
        with self._lock:
            if model in self._limiters:
                return self._limiters[model]
            # Check provider prefix (e.g., "anthropic/" matches "anthropic/claude-...")
            for prefix, limiter in self._limiters.items():
                if model.startswith(prefix):
                    return limiter
            return None


# Global registry instance
rate_limiters = RateLimiterRegistry()
```

**Key design decisions:**
- `_async_lock` is `asyncio.Lock`, not `threading.Lock`, to avoid deadlocks from holding a thread lock across `await asyncio.sleep()`
- `_async_semaphore` is created eagerly in `__init__` (not lazily) to avoid race conditions from concurrent coroutines
- `arelease()` is a sync method (not `async def`) because `asyncio.Semaphore.release()` is synchronous — the `a` prefix indicates it pairs with `aacquire`, not that it's a coroutine

**Integration into `utils.py`:**

```python
from .rate_limit import rate_limiters

def get_model_response(messages, model=..., rate_limiter=None, ...):
    limiter = rate_limiter or rate_limiters.get(model)
    if limiter is not None:
        limiter.acquire()
```

```python
class AudioModel:
    """Audio transcription via OpenAI-compatible /audio/transcriptions endpoint."""

    def __init__(
        self,
        model: str = "whisper-1",
        api_key: str = None,
        base_url: str = None,
        language: Optional[str] = None,
        response_format: str = "json",      # json, text, srt, verbose_json, vtt
        counter: Optional[NerifTokenCounter] = None,
    ):
        self.model = model
        self.api_key = api_key or _OPENAI_API_KEY
        self.base_url = (base_url or _OPENAI_API_BASE).rstrip("/")
        self.language = language
        self.response_format = response_format
        self.counter = counter

    def transcribe(
        self,
        file: Union[str, Path, BinaryIO],
        prompt: Optional[str] = None,
        language: Optional[str] = None,
        response_format: Optional[str] = None,
        temperature: Optional[float] = None,
    ) -> dict:
        """Transcribe audio file.

        Args:
            file: Path to audio file, or file-like object, or (filename, bytes) tuple.
            prompt: Optional prompt to guide transcription.
            language: ISO-639-1 language code (overrides instance default).
            response_format: Override instance default format.
            temperature: Sampling temperature (0-1).

        Returns:
            API response dict. For json format: {"text": "..."}.
            For verbose_json: includes segments, language, duration.
        """
        url = f"{self.base_url}/audio/transcriptions"
        headers = {"Authorization": f"Bearer {self.api_key}"}

        # Build file argument and data
        data = {"model": self.model}
        if prompt is not None:
            data["prompt"] = prompt
        lang = language or self.language
        if lang is not None:
            data["language"] = lang
        fmt = response_format or self.response_format
        if fmt is not None:
            data["response_format"] = fmt
        if temperature is not None:
            data["temperature"] = str(temperature)

        # Open file inside try block to prevent handle leaks
        opened_file = None
        try:
            if isinstance(file, tuple):
                files = {"file": file}
            elif isinstance(file, (str, Path)):
                opened_file = open(file, "rb")
                files = {"file": (str(file), opened_file)}
            else:
                files = {"file": ("audio", file)}

            resp = httpx.post(url, headers=headers, files=files, data=data, timeout=_DEFAULT_TIMEOUT)
            resp.raise_for_status()
        finally:
            if opened_file is not None:
                opened_file.close()

        return resp.json()

    async def atranscribe(self, file, prompt=None, language=None,
                           response_format=None, temperature=None) -> dict:
        """Async version of transcribe(). Same parameters, uses httpx.AsyncClient."""
        url = f"{self.base_url}/audio/transcriptions"
        headers = {"Authorization": f"Bearer {self.api_key}"}

        data = {"model": self.model}
        if prompt is not None:
            data["prompt"] = prompt
        lang = language or self.language
        if lang is not None:
            data["language"] = lang
        fmt = response_format or self.response_format
        if fmt is not None:
            data["response_format"] = fmt
        if temperature is not None:
            data["temperature"] = str(temperature)

        opened_file = None
        try:
            if isinstance(file, tuple):
                files = {"file": file}
            elif isinstance(file, (str, Path)):
                opened_file = open(file, "rb")
                files = {"file": (str(file), opened_file)}
            else:
                files = {"file": ("audio", file)}

            async with httpx.AsyncClient(timeout=_DEFAULT_TIMEOUT) as client:
                resp = await client.post(url, headers=headers, files=files, data=data)
                resp.raise_for_status()
        finally:
            if opened_file is not None:
                opened_file.close()

        return resp.json()

    def translate(self, file: Union[str, Path, BinaryIO], prompt: Optional[str] = None) -> dict:
        """Translate audio to English text via /audio/translations endpoint."""
        url = f"{self.base_url}/audio/translations"
        headers = {"Authorization": f"Bearer {self.api_key}"}
        data = {"model": self.model}
        if prompt is not None:
            data["prompt"] = prompt

        opened_file = None
        try:
            if isinstance(file, (str, Path)):
                opened_file = open(file, "rb")
                files = {"file": (str(file), opened_file)}
            else:
                files = {"file": ("audio", file)}
            resp = httpx.post(url, headers=headers, files=files, data=data, timeout=_DEFAULT_TIMEOUT)
            resp.raise_for_status()
        finally:
            if opened_file is not None:
                opened_file.close()
        return resp.json()

    async def atranslate(self, file, prompt=None) -> dict:
        """Async version of translate()."""
        url = f"{self.base_url}/audio/translations"
        headers = {"Authorization": f"Bearer {self.api_key}"}
        data = {"model": self.model}
        if prompt is not None:
            data["prompt"] = prompt

        opened_file = None
        try:
            if isinstance(file, (str, Path)):
                opened_file = open(file, "rb")
                files = {"file": (str(file), opened_file)}
            else:
                files = {"file": ("audio", file)}
            async with httpx.AsyncClient(timeout=_DEFAULT_TIMEOUT) as client:

            if opened_file is not None:

        return resp.json()
```

```python
class SpeechModel:
    """Text-to-speech via OpenAI-compatible /audio/speech endpoint."""

    VOICES = ("alloy", "ash", "coral", "echo", "fable", "onyx", "nova", "sage", "shimmer")

    def __init__(
        self,
        model: str = "tts-1",
        api_key: str = None,
        base_url: str = None,
        voice: str = "alloy",
        response_format: str = "mp3",    # mp3, opus, aac, flac, wav, pcm
        speed: float = 1.0,              # 0.25 to 4.0
        counter: Optional[NerifTokenCounter] = None,
    ):
        self.model = model
        self.api_key = api_key or _OPENAI_API_KEY
        self.base_url = (base_url or _OPENAI_API_BASE).rstrip("/")
        self.voice = voice
        self.response_format = response_format
        self.speed = speed
        self.counter = counter

    def text_to_speech(
        self,
        text: str,
        voice: Optional[str] = None,
        model: Optional[str] = None,
        response_format: Optional[str] = None,
        speed: Optional[float] = None,
    ) -> bytes:
        """Generate speech audio from text.

        Returns: Raw audio bytes in the requested format.
        """
        url = f"{self.base_url}/audio/speech"
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json",
        }
        body = {
            "model": model or self.model,
            "input": text,
            "voice": voice or self.voice,
            "response_format": response_format or self.response_format,
            "speed": speed or self.speed,
        }
        resp = httpx.post(url, json=body, headers=headers, timeout=_DEFAULT_TIMEOUT)
        resp.raise_for_status()
        return resp.content

    async def atext_to_speech(self, text, voice=None, model=None,
                               response_format=None, speed=None) -> bytes:
        """Async version of text_to_speech()."""
        url = f"{self.base_url}/audio/speech"
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json",
        }
        body = {
            "model": model or self.model,
            "input": text,
            "voice": voice or self.voice,
            "response_format": response_format or self.response_format,
            "speed": speed or self.speed,
        }
        async with httpx.AsyncClient(timeout=_DEFAULT_TIMEOUT) as client:
            resp = await client.post(url, json=body, headers=headers)
            resp.raise_for_status()
        return resp.content

    def text_to_speech_file(
        self,
        text: str,
        output_path: Union[str, Path],
        **kwargs,
    ) -> Path:
        """Generate speech and save to file. Returns the output path."""
        audio_bytes = self.text_to_speech(text, **kwargs)
        path = Path(output_path)
        path.write_bytes(audio_bytes)
        return path

    async def atext_to_speech_file(self, text, output_path, **kwargs) -> Path:

        audio_bytes = await self.atext_to_speech(text, **kwargs)
        path = Path(output_path)
        path.write_bytes(audio_bytes)
        return path
```

## 9. Technical Debt

### 9a. Python Version Inconsistency
- `README.md` says `>= 3.9`, `pyproject.toml` says `>=3.10`
- **Fix**: Update README to `>= 3.10` (the codebase uses `int | None` union syntax which requires 3.10+)

### 9b. Empty ASR/TTS Optional Dependencies
- `pyproject.toml` has `asr = []` and `tts = []` (empty dep lists)
- **Fix**: Keep them empty — they use only `httpx` which is already a core dependency. The optional groups exist for documentation/discoverability, not for extra packages.

### 9c. Keywords Field
- `pyproject.toml` has `keywords = ["feed", "reader", "tutorial"]` — clearly wrong
- **Fix**: Update to `keywords = ["llm", "ai", "natural-language", "python", "nerif"]`

### 9d. Agent Parallel Tool Calls (async only)
- Currently the agent processes tool calls sequentially even when the model returns multiple tool calls in one response
- **Fix**: In `arun()`, execute multiple tool calls concurrently with `asyncio.gather()` when there are multiple tool calls in a single response (see section 1 for implementation). The sync `run()` keeps sequential execution for simplicity.

### 9e. Clean up asr/tts re-exports
- `asr/__init__.py` and `tts/__init__.py` re-export `SimpleChatModel` and `MultiModalMessage` which belong to `nerif.model`
- **Fix**: Remove these re-exports (see sections 7 and 8). Users should import from `nerif.model` directly.
